# Meatguy Steakhouse — Landing Page (One-Pager)

## Tujuan
Halaman landing sederhana untuk mengarahkan traffic dari iklan video Meatguy. Bukan web app, hanya satu halaman dengan satu tujuan utama: **mendorong pengunjung untuk menghubungi via WhatsApp** (reservasi / inquiry).

## Referensi branding
- Website resmi: https://meatguysteakhouse.com/
- Branding: **premium** — tidak boleh terlihat murah atau generic.
- Tone: passion, craftsmanship, dry-aged premium cuts, "Where Passion Meets Perfection".

## Persyaratan
- **Tipe:** One-pager (single page), bukan aplikasi dengan banyak fitur.
- **Fitur:** Tidak perlu fitur aneh (no login, no complex forms). Fokus ke pengalaman baca + satu CTA.
- **Pengalaman:** Interactive dan immersive — scroll yang enak, mungkin sedikit motion/parallax, typography dan visual yang bikin betah.
- **CTA utama:** Contact / Reservasi via **WhatsApp** — tombol atau link yang jelas ke WA.
- **Nomor WhatsApp:** +62 812-5007-0094 (dari website resmi).

## Konten yang bisa dipakai (dari website resmi)
- Tagline: "Where Passion Meets Perfection"
- World's 101 Best Steak Restaurants — No. 90 (2025)
- Lokasi: Meatguy Bintaro & Meatguy SCBD
- Founder: Dimas Ramadhan Pangestu (Dimsthemeatguy); Chef: Teuku Wahyudi (El Patrón)
- Cukup ringkas; tidak perlu semua detail — cukup buat orang tertarik dan mau chat WA.

## Tech
- Standalone (satu file HTML + CSS + JS boleh, atau minimal build) supaya bisa di-deploy di mana saja (Vercel, Netlify, atau static host).
- Mobile-friendly, cepat load.

## Deliverable
Satu halaman yang:
1. Terasa premium dan selaras dengan meatguysteakhouse.com
2. Interactive/immersive (scroll, mungkin subtle animation)
3. Ujung-ujungnya mengarah ke satu CTA jelas: **Chat / Reservasi via WhatsApp**
